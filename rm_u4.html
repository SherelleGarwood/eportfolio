Literature Review Outline

Literature review outline: Implementing Machine Learning tools and techniques in crime prediction.

The literature review will be focussed upon the implementation of machine learning tools and techniques in crime prediction. The review will analyse literature and studies that have used machine learning algorithms, such as decision trees to predict crime. The review will also aim to analyse the effectiveness of these tools and techniques as measures in the prevention of crime. Another important factor is the ethical factors regarding machine learning and the possible ramifications of this.

I will be taking the perspective (based on current reading) that machine learning can be a useful aid for crime prevention but should be used in conjunction with traditional methods.

Structure of my review:

    1. Introduction to machine learning tools and techniques
    2. Brief overview of the use of machine learning in crime prediction
    3. Review of literature/studies using machine learning algorithms
      1. Vector machines- Gerber and Johnson
      2. Neural networks- Bowers et al.
      3. Decision trees- Mohler et al.
      4. Kim et al
      5. Shah et al
    4. Effectiveness of machine learning within crime prevention
    5. Potential drawbacks of machine learning within crime prevention
    6. Conclusion
